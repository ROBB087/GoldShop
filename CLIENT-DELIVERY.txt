GoldShop Client Delivery
========================

Recommended file to give the client:
- GoldShop-Setup.exe

Alternative manual package:
- all files in this same folder

Client activation steps after installation:
1. Open GoldShop from the desktop or Start Menu shortcut
2. Copy the Machine ID shown in the activation window
3. Send the Machine ID to the software owner
4. Paste the activation key received from the owner
5. Press activate

Important:
- The installer places the application under the current Windows user profile
- Application data, backups, and logs are stored automatically in %LocalAppData%\GoldShop
- The activation key is valid only for this device
- If the client changes device, a new activation key is required
- Do not delete license files after activation
